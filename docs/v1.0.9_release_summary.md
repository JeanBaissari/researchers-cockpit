# v1.0.9 Release Summary

**Date:** 2026-01-10

## Overview

Version 1.0.9 introduces new trading utility modules, enhanced strategy template, comprehensive test coverage, and development infrastructure improvements. This release builds upon the solid modular architecture established in v1.0.8.

---

## What's New

### 1. New Trading Utility Modules

Three new reusable utility modules added to `lib/`:

| Module | Purpose | Lines |
|--------|---------|-------|
| `lib/pipeline_utils.py` | Pipeline API setup and validation | 187 |
| `lib/position_sizing.py` | Position sizing calculations (fixed, volatility-scaled, Kelly) | 250 |
| `lib/risk_management.py` | Exit condition checks (stop loss, trailing stop, take profit) | 234 |

**Total:** 671 lines of new reusable functionality

**Features:**
- Pipeline validation with asset class compatibility checks
- Multiple position sizing methods (fixed, volatility-scaled, Kelly Criterion)
- Comprehensive risk management (fixed stop, trailing stop, take profit)
- Full type hints and documentation
- Follows SRP (Single Responsibility Principle)

### 2. Enhanced Strategy Template

**File:** `strategies/_template/strategy.py`

**Improvements:**
- Organized imports following project standards (stdlib, third-party, local)
- Integrated new utility modules (pipeline_utils, position_sizing, risk_management)
- Improved code organization and maintainability
- Better documentation and examples

**File:** `strategies/_template/parameters.yaml`

**Additions:**

- Signal generation parameters

- Timezone handling guidelines
